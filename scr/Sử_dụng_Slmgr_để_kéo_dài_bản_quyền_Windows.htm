1. Mở Command Prompt dưới quyền Admin

Trên Windows 8 hoặc Windows 10 bạn có thể kích chuột phải vào nút Start hoặc nhấn tổ hợp phím Windows + X, sau đó chọn Command Prompt (Admin).

Trên Windows 7, bạn gõ từ khóa Command Prompt vào khung Search trên Menu Start sau đó kích chuột phải vào Command Prompt chọn Run as Administrator.

2. Kiểm tra thông tin Activation, bản quyền (License) và ngày hết hạn (Expiration Date)

Để hiển thị các thông tin bản quyền, kích hoạt trên hệ thống hiện tại, bạn nhập câu lệnh dưới đây vào cửa sổ Command Prompt:

slmgr.vbs /dli

Nếu muốn hiển thị chi tiết thông tin bản quyền - bao gồm mã Activation ID, mã Installation ID và một số thông tin chi tiết khác, bạn sử dụng câu lệnh:

slmgr.vbs /dlv

3. Kiểm tra ngày hết hạn bản quyền

Để hiển thị ngày hết hạn bản quyền, bạn sử dụng câu lệnh dưới đây:

slmgr.vbs /xpr

Lưu ý:

Câu lệnh chỉ hữu ích trên hệ thống Windows được kích hoạt qua server KMS (máy chủ KMS). Các phiên bản Retail và nhiều phiên bản khác phải có bản quyền vĩnh viễn, không hết hạn.

Nếu bạn không được cung cấp Product Key, nó sẽ hiển thị thông báo lỗi trên màn hình.

3. Gỡ bỏ Product Key

Bạn có thể sử dụng Slmgr để gỡ bỏ Product Key trên phiên bản Windows hiện tại.

Nhập câu lệnh dưới đây vào cửa sổ Command Prompt rồi tiến hành khởi động lại máy tính của bạn:

slmgr.vbs /upk

Ngoài ra Windows cũng lưu trữ Product Key trên Registry, trong một số trường hợp cần thiết Windows sẽ sử dụng key trên Registry để cài đặt máy tính của bạn. Do đó bạn cũng phải tiến hành gỡ bỏ Product Key trên Registry.

Sử dụng câu lệnh dưới đây để gỡ bỏ Product Key trên Registry:

slmgr.vbs /cpky

Việc chạy một câu lệnh trên không gỡ bỏ cài đặt Product Key được. Nó chỉ gỡ bỏ Product Key trên Registry để hệ thống không thể truy cập được trên Registry.

4. Thiết lập hoặc thay đổi Product Key

Bạn có thể sử dụng slmgr.vbs để nhập một Product Key mới. Nếu hệ thống Windows của bạn đã có Product Key, bạn có thể sử dụng câu lệnh dưới đây để thay thế Product Key cũ bằng một Product Key mới:

slmgr.vbs /ipk #####-#####-#####-#####-#####

Lưu ý:

Thay thế #####-#####-#####-#####-##### bằng Product Key.

Câu lệnh sẽ kiểm tra Product Key mà bạn nhập có trùng với Product Key trước hay không.

Sau khi chạy câu lệnh xong, Microsoft sẽ hiển thị thông báo khởi động lại máy tính của bạn.

Ngoài ra bạn cũng có thể thay đổi Product Key trong ứng dụng Settings app trên giao diện Activation.

5. Active Windows Online

Để buộc Windows cố gắng Active Online, bạn sử dụng câu lệnh dưới đây:

slmgr.vbs /ato

Nếu bạn đang sử dụng phiên bản Retail, câu lệnh sẽ buộc Windows cố gắng online với Server Micorsoft (máy chủ Microsoft).

Nếu hệ thống được thiết lập sử dụng máy chủ KMS để active, nó sẽ cố gắng active với máy chủ KMS trên hệ thống mạng nội bộ (Local Network). Lệnh này có thể hữu ích nếu Windows không kích hoạt bằng một kết nối hoặc các vấn đề máy chủ.

6. Active Windows Offline

Để nhận mã Installation ID để Active Offline, bạn sử dụng câu lệnh dưới đây:

slmgr.vbs /dti

Tiếp theo bạn cần môt ID xác nhận để kích hoạt. Bạn có thể gọi đến trung tâm kích hoạt của Microsoft để được cung cấp mã Installation ID.

Để nhập ID xác nhận mà bạn vừa nhận, bạn sử dụng câu lệnh dưới đây:

slmgr.vbs /atp ACTIVATIONID

Lưu ý:

Thay thế ACTIVATIONID bằng mã ID kích hoạt mà bạn nhận được.

Sau khi thực hiện xong, bạn có thể sử dụng câu lệnh: slmgr.vbs /dli or slmgr.vbs /dlv commands để xác nhận bạn đã kích hoạt.

7. Kéo dài thời gian dùng thử

Một số phiên bản Windows thường giới hạn thời gian sử dụng bản thử nghiệm miễn phí.Chẳng hạn như Windows 7 cung cấp 30 ngày dùng thử.

Để kéo dài thời gian dùng thử bạn có thể sử dụng câu lệnh dưới đây:

slmgr.vbs /rearm

Lệnh này chỉ có thể được sử dụng nhiều lần, vì vậy bạn không thể kéo dài thời gian sử dụng bản dùng thử không giới hạn.

Số lượng thời gian bạn có thể sử dụng bản dùng thử phụ thuộc vào "rearm count". Bạn có thể xem bằng cách sử dụng lệnh:

slmgr.vbs /dlv

8. Điều khiển máy tính của bạn từ xa

Slmgr thường thực hiện các hành động mà bạn chỉ định trên máy tính hiện tại.

Tuy nhiên, bạn cũng có thể sử dụng Slmgr để quản lý từ xa các máy tính trên hệ thống mạng của bạn nếu bạn có quyền truy cập các máy tính đó.

Lệnh dưới đây để thực hiện các hành động mà bạn chỉ định trên máy tính hiện tại:

slmgr.vbs /option

Lệnh dưới đây để điều khiển một máy tính từ xa:

slmgr.vbs computername username password /option

Lưu ý:

Bạn phải có tên máy tính, tên người dùng và mật khẩu.

Thay thế computername username password bằng tên máy tính, tên người dùng và mật khẩu.
